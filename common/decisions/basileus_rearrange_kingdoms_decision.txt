# =====================================================================
# BASILEUS — REARRANGE THE KINGDOMS (all KINGDOM-tier reorganizations)
# ---------------------------------------------------------------------
# The single home for KINGDOM-tier de jure reshaping (the strict tier split;
# duchy-tier merges live in basileus_reorganize_themes_decision). The
# e_byzantium holder, having fully secured a region, may form a theme-kingdom
# from its duchies, dissolve rival crowns into one, or restore a lost diadem of
# the Roman world. One-time per region (locked via the unavailable_unique_decisions
# global list). Uses CK3's vanilla decision option-list widget.
#
# Effects, loc descriptions, regions and lock flags are SHARED with the themes
# system (basileus_theme_effects.txt / basileus_themes_l_english.yml / regions);
# only the decision-side wiring lives here. The Anatolia widget is special — it
# requires the four Anatolian duchy merges first and runs a rename/split/grant
# event chain.
# =====================================================================
