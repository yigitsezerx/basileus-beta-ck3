# =====================================================================
# BASILEUS — REORGANIZE THE THEMES (duchy-tier reorganizations only)
# ---------------------------------------------------------------------
# A multi-option (widget) decision: the e_byzantium holder, having fully
# secured a frontier region, consolidates it into a historical theme by
# merging the region's duchies into one surviving DUCHY, renaming it, and
# destroying the emptied parent duchies. One-time per theme (locked via the
# unavailable_unique_decisions global list).
#
# KINGDOM-tier reorganizations live in basileus_rearrange_kingdoms_decision
# (the strict tier split). This decision is duchy-tier only.
#
# Adding a duchy theme = one item below + one basileus_*_effect +
# one lock flag + its loc keys.
# =====================================================================
